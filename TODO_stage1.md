# 番茄学习法软件 — 功能开发任务清单
## 第一阶段：MVP 核心功能（必须优先完成）
### 1. 核心计时器模块
- [ ] 实现 **25分钟专注计时**（默认）
- [ ] 实现 **5分钟短休息计时**
- [ ] 实现 **15分钟长休息计时**
- [ ] 实现计时器控制：开始、暂停、重置、跳过
- [ ] 实现番茄周期逻辑：每完成4个专注后自动进入长休息
- [ ] 时间显示支持：分钟/秒双模式，支持全屏专注模式
- [ ] 计时结束时自动切换下一阶段（可配置是否自动衔接）

### 2. 任务管理模块
- [ ] 实现 **今日任务列表**：添加、编辑、删除任务
- [ ] 为每个任务设置 **预估番茄数**
- [ ] 计时器运行时 **绑定当前任务**
- [ ] 任务状态切换：待办 / 进行中 / 已完成
- [ ] 记录每个任务 **实际消耗的番茄数**

### 3. 提醒与通知模块
- [ ] 实现 **音效提示**：专注开始/结束、休息结束时的提示音
- [ ] 实现 **桌面系统通知**（Windows/macOS/Linux原生通知）
- [ ] 支持 **静音模式**（仅通知不发声）
- [ ] 支持 **窗口闪烁或置顶提醒**

### 4. 本地数据持久化
- [ ] 实现 **本地数据存储**（SQLite或JSON文件）
- [ ] 保存每次番茄的：开始时间、结束时间、关联任务、是否完成
- [ ] 保存任务列表及状态
- [ ] 支持 **数据导入/导出备份**
